Virtual Schema Installer is a Command Line Interface (CLI) tool that automates a Virtual Schema installation.

## Features

Installs the following Virtual Schema to the Exasol Database:

* DB2
* ElasticSearch
* Microsoft SQL Server
* MySQL
* Oracle
* PostgreSQL
